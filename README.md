## 项目运行

### `API Server`启动

```sh
# 将项目克隆到本地
git clone https://github.com/hechen77/CarbonStore.git

# 进入项目
cd ./CarbonStore

# 进入API项
cd ./CarbonStoreAPI
# 安装依赖
npm install
# 启动API
npm run serve
```
